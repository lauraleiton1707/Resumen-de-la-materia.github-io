Resumen: Ingeniería de Software

Metodologías de Desarrollo de Software

Las metodologías son guías o enfoques organizados que se utilizan para planificar, estructurar y controlar el desarrollo de sistemas de software. Algunas destacadas son:

  • Cascada: Modelo secuencial en el que cada etapa debe completarse antes de avanzar a la siguiente.
  • Ágil: Proceso iterativo con entregas incrementales, orientado a la retroalimentación constante de los stakeholders.
  • Scrum: Un marco dentro de Agile que utiliza roles (como Scrum Master), artefactos y reuniones diarias para gestionar proyectos en sprints.
  • Kanban: Herramienta visual para gestionar el flujo de trabajo con tableros y tarjetas.

Comandos Básicos de Git

Git es un sistema de control de versiones distribuido ampliamente usado en el desarrollo de software. Los siguientes comandos permiten gestionar repositorios en GitHub:

# Inicializar un repositorio
git init

# Agregar archivos al área de preparación
git add .

# Crear un commit con un mensaje
git commit -m "Comentario del commit"

# Vincular el repositorio local con GitHub
git remote add origin 

# Subir los cambios a GitHub
git push -u origin main

# Actualizar el repositorio local con cambios del remoto
git pull origin main

Código Python para Base de Datos SQLite

A continuación, se presenta un código en Python para crear una base de datos llamada "autoconocimiento.db" y una tabla relacionada:

import sqlite3 as sql

# Crear la base de datos
def createDB():
    conn = sql.connect("autoconocimiento.db")
    print("Base de datos creada")
    conn.close()

# Crear la tabla "respuestas"
def createTable():
    conn = sql.connect("autoconocimiento.db")
    cursor = conn.cursor()
    cursor.execute("""
       CREATE TABLE respuestas (
           id_respuesta INTEGER PRIMARY KEY AUTOINCREMENT,
           id_pregunta INTEGER,
           respuesta_usuario TEXT CHECK(respuesta_usuario IN ('A', 'B')) NOT NULL,
           FOREIGN KEY (id_pregunta) REFERENCES preguntas(id_pregunta)
       );
    """)
    conn.commit()
    conn.close()

if _name_ == "_main_":
    createDB()
    createTable()

Ejemplo de Tabla para la Base de Datos

La estructura de la tabla respuestas queda de la siguiente manera:

Campo Tipo Descripción
